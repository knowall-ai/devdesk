# DevDesk

DevDesk is a help desk and customer support platform inspired by Zendesk. This project aims to provide a comprehensive ticketing system, knowledge base, and customer support tools.

## Project Status

This project is in the planning phase. See the [Feature Issues](.github/ISSUES/README.md) directory for a comprehensive list of features to be implemented.

## Getting Started

### Creating GitHub Issues

To create all planned features as GitHub issues, run:

```bash
# Ensure you have GitHub CLI installed and authenticated
gh auth login

# Run the issue creation script
bash .github/ISSUES/create_issues.sh
```

This will create:
- 15 labels for categorization
- 66 feature issues with detailed requirements
